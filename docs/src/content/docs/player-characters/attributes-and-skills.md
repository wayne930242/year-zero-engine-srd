## 玩家角色

你的玩家角色（PC）是任何元年引擎遊戲中最重要的資產。他們是你的化身，是你在世界中的眼睛與耳朵。但反過來，他們也依賴你為他們做出正確的決策。認真對待你的玩家角色，把他們當作真實的人來扮演——這樣才會更有趣。與此同時，也不要試圖讓你的玩家角色遠離每一個可以預見的危險。遊戲的目標是創造一個精彩的故事，而這需要你承擔風險。

在遊戲的過程中，你的玩家角色將會成長與發展。他們的技能和專長可以透過經驗習得，你也會發現他們的性格如何在無法以數字衡量的方式中改變與形成。這正是你的玩家角色真正活起來的時刻。

## 角色卡

要創建你的玩家角色，你需要一張角色卡。本 SRD 不提供角色卡，因為任何元年引擎遊戲都需要根據特定規則版本和遊戲背景量身訂製的角色卡。

## 原型

大多數元年引擎遊戲都提供某種類型的角色原型供選擇。這些原型——也可稱為「職業」、「身份」或「生涯」——以遊戲世界為基礎，幫助玩家掌握遊戲背景。

原型決定了你是什麼樣的人、你的背景以及你在團隊中的角色。你的原型將影響你的屬性、技能、專長和起始裝備。原型可能感覺有些刻板，但這本就是它的用意。選擇一個原型，是讓你自己和其他玩家立即感受你角色特質的快速方式。但請記住，你的角色不僅僅是他們的原型——原型只是創造一個獨特玩家角色的起點。

**關鍵屬性：** 在許多元年引擎遊戲中，每個原型都有一個「關鍵屬性」。你的關鍵屬性最高可達 5 點，其他屬性則上限為 4 點。

## 屬性

你的角色擁有四項屬性，代表你基本的身體與心智能力。屬性在你擲骰執行行動時使用，同時也決定了你在崩潰之前能承受多少傷害與壓力。詳見第 4 章。

- **力量：** 原始的肌肉力量與體魄
- **敏捷：** 身體控制、速度與動作技巧
- **機智：** 感知、智力與精神狀態
- **同理：** 個人魅力與影響他人的能力

:::note[骰池屬性]
在骰池版本的元年引擎中，人類屬性值的範圍為 1 到 5，詳見下表。

**起始分數：** 一般情況下，你可以將 14 點分配到各屬性中。每項屬性最少分配 2 點，最多 5 點。

| 屬性值 | 描述 |
| ------ | ---- |
| 5 | 超凡 |
| 4 | 優秀 |
| 3 | 普通 |
| 2 | 低於平均 |
| 1 | 虛弱 |
:::

:::note[階梯骰屬性]
在階梯骰版本的元年引擎中，屬性以 A 到 D 的等級評定，每個等級對應特定的骰型，詳見下表。

**起始分數：** 所有四項屬性的基準值均為 C。你可以進行三次提升，每次提升一階，最高提升至 A。你可以自由選擇提升哪些屬性。你可以將一項屬性從 C 降至 D，以換取一次額外的提升機會。

**骰面數：** 階梯骰規則中有時會使用「骰面數」這個詞，它單純指某種骰型上最高可能出現的點數。D6 的骰面數為 6，D8 的骰面數為 8，以此類推。

| 屬性等級 | 骰型 | 骰面數 | 描述 |
| -------- | ---- | ------ | ---- |
| A | D12 | 12 | 超凡 |
| B | D10 | 10 | 優秀 |
| C | D8 | 8 | 普通 |
| D | D6 | 6 | 虛弱 |
:::

## 生命值與意志

你的屬性決定了你在崩潰並退出行動之前能承受多少傷害與壓力。在某些元年引擎遊戲中，這以生命值和意志分數來衡量。詳見第 4 章。

:::note[骰池計算方式]
- **生命值：** 起始生命值等於力量與敏捷分數的平均值（向上取整），再加一。
- **意志：** 起始意志等於機智與同理分數的平均值（向上取整），再加一。
:::

:::note[階梯骰計算方式]
- **生命值：** 起始生命值等於力量與敏捷骰面數之和除以 4（向上取整）。
- **意志：** 起始意志等於機智與同理骰面數之和除以 4（向上取整）。
:::

:::note[變體規則]
元年引擎中還有其他管理傷害與創傷的方式，不一定需要生命值或意志分數。以下是兩種變體：

- **僅生命值：** 某些元年引擎遊戲只有生命值，沒有意志。
- **屬性傷害：** 傷害直接作用於屬性，降低其效能。若任何屬性降至零，你便崩潰。
- **狀態：** 每點傷害給予你一個狀態（第 21 頁），當你承受的狀態達到設定數量時即崩潰。
:::

## 技能

你的技能是你在人生中習得的知識與能力。技能非常重要，因為它們與屬性一同決定了你在遊戲中執行特定行動的效率。元年引擎 SRD 共有十二項基本技能，全部在第 3 章詳細說明。部分元年引擎遊戲擁有不同或額外的技能。

:::note[骰池技能]
在骰池版本的元年引擎中，技能以 0 到 5 的等級衡量，數字越高越好。

**沒有技能等級？** 即使你在某項技能上沒有等級，你仍然可以擲骰——在此情況下，你只使用該技能對應的屬性和裝備進行擲骰。詳見下一章的技能說明。

**起始技能：** 一般情況下，你可以將 10 點分配到起始技能中。在許多元年引擎遊戲中，每個原型都列出若干關聯技能。你在原型技能的起始等級最高為 3，其他所有技能的起始等級上限為 1。

| 技能等級 | 描述 |
| -------- | ---- |
| 5 | 精英 |
| 4 | 老練 |
| 3 | 經驗豐富 |
| 2 | 受過訓練 |
| 1 | 新手 |
:::

:::note[階梯骰技能]
在階梯骰版本的元年引擎中，技能等級與屬性相同，以 A 到 D 的等級衡量，每個等級對應特定的骰型，詳見下表。

**沒有技能等級？** 即使你在某項技能上完全沒有等級，通常仍可擲骰——在此情況下，只使用對應屬性的骰型。

**起始技能：** 選擇一項 B 等級技能、兩項 C 等級技能，以及三項 D 等級技能。你的 B 等級技能通常必須是原型所列出的技能之一。C 和 D 等級技能可自由選擇。

| 技能等級 | 骰型 | 骰面數 | 描述 |
| -------- | ---- | ------ | ---- |
| A | D12 | 12 | 精英 |
| B | D10 | 10 | 老練 |
| C | D8 | 8 | 經驗豐富 |
| D | D6 | 6 | 新手 |
:::

:::note[角色創建七步驟]
本章詳細說明如何創建你的玩家角色，以下摘要提供便於參考的概覽。準備好角色卡和鉛筆，按照以下步驟進行：

1. 選擇你的原型。
2. 決定你的屬性。
3. 決定你的技能。
4. 選擇你的起始專長。
5. 決定你的性格特質。
6. 選擇你的裝備。
7. 取一個名字。
:::
