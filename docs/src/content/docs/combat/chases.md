## 追逐
本章先前的規則主要處理兩方之間的戰鬥。另一種不同型態的對抗則是追逐。追逐可以徒步進行，也可以使用載具。

## 開始追逐
追逐可能直接從角色扮演場景開始，也可能是在戰鬥後有人逃跑時接續發生。無論是哪種情況，主持人都會宣告追逐開始，並以[地圖、區域與先攻](/combat/maps-and-initiative/)的距離類別表為參考，決定參與者之間的起始距離，最遠不超過長程。

在追逐中，距離只用距離類別衡量，不使用區域。大多數追逐只有兩方，也就是追擊者與獵物。若要處理多名參與者，請見下方多名參與者規則。

## 結算
追逐和戰鬥一樣以輪進行，但不抽先攻，也不執行一般動作。相對地，每名參與者在每輪都會秘密選擇一個追逐動作，之後同時揭露；而主持人則會隨機產生障礙來阻擋他們。障礙可以用下方的表格隨機擲出。每輪追逐依照下列流程進行：

1. 每名參與者秘密選擇一個動作。
2. 主持人隨機產生一個障礙並揭露。
3. 追擊者與獵物同時揭露自己的動作。
4. 動作依序結算，由獵物先結算，追擊者最後。

玩家角色或非玩家角色可以取消原本規劃的動作，但不能改選其他動作；該參與者該輪就只是什麼都不做。

### 徒步對載具
如果追逐中的一方在奔跑，另一方在載具裡，追逐規則仍照常運作，只需使用各自適用的技能即可。在繁忙街道環境中，開車未必比較快，但駕駛會更安全。不過，在較開闊、載具速度優勢能充分發揮的環境中，載具中的那一方在與徒步對手進行追逐時，所有追擊／逃離擲骰都獲得 +2 調整值。

## 追逐動作
在追逐的每一輪中，每名參與者都可以執行一個動作，所有動作如下所述。這些動作必須秘密選擇，可以用卡片，也可以直接寫在紙上。

動作通常需要技能擲骰。這些擲骰可能會受到主持人在動作結算前揭露的追逐障礙影響。若使用載具，會以方括號標示對應的技能或因素。

### 追擊／逃離
你只是盡可能快地奔跑或駕駛。擲機動〔若你的遊戲有這項技能，則改擲駕駛〕。每擲出一個成功，與另一名參與者的距離就會縮短一個距離類別（若你是追擊者），或增加一個距離類別（若你是獵物）。

- 若在本輪雙方動作都結算完之後，距離超過極遠程，追逐就結束。  
- 若在本輪雙方動作都結算完之後，距離變成貼身（或更近），追擊者可以立刻進行一次近戰攻擊〔載具則是撞擊攻擊〕。距離維持在貼身。  
- 若距離原本會降到貼身以下，而追擊者選擇不依上文發動攻擊，則可以選擇維持貼身距離，或超越獵物，改由自己變成獵物，並依超出的成功數決定距離（一個超出成功代表短程，依此類推）。後者通常只適用於賽跑一類的情境。  

### 躲藏
你迅速鑽進小巷，或躲到某個東西後面。這個動作只能由獵物執行，而且不能在短程或更近時使用。若目前距離是中程，你承受 −2 調整值；若距離是極遠程，則獲得 +2 調整值。

以潛行〔駕駛〕對抗追擊者的觀察進行一次對抗擲骰。若你贏得擲骰，追逐就結束；或者你也可以選擇對追擊者發動一次偷襲，距離由你自行決定。

### 阻擋
你把身後的東西撞翻，試圖替追擊者製造障礙。這個動作只能由獵物執行。擲強力〔駕駛〕。若成功，你與追擊者的距離增加一個距離類別，而追擊者在結算自己本輪動作之前，也必須先擲強力〔駕駛〕。若這次擲骰失敗，他們的動作就會被取消。

### 攔截
你試圖找捷徑，截住獵物。這個動作只能由追擊者執行。進行一次機動〔駕駛〕對抗擲骰。若距離為長程或極遠程，你承受 −2 調整值。若你贏得擲骰，與獵物的距離會立刻縮短到貼身，並可依上文立刻進行一次近戰／撞擊攻擊。若你輸掉或平手，距離就增加到極遠程；如果原本已經是極遠程，追逐就直接結束。

### 停步射擊
你停下來吸一口氣，瞄準武器後開火。這就照一般遠程武器攻擊處理。

### 其他
你做出不屬於上述任何動作的事。一般來說，這表示本輪你相對於另一名參與者沒有移動。最終由主持人裁定。

## 多名參與者
若追逐中有超過兩名參與者，有兩種處理方式。最簡單的方式是把參與者分成兩個陣營。每一方只由最慢的那個人／載具，也就是在套用機動性調整值後技能等級最低者（若平手則自由選定）來執行動作與擲骰，唯一例外是「停步射擊」，此時該方所有參與者都可以執行。

另一種較複雜的方式，是讓每名參與者都各自執行動作。這時你必須追蹤每名參與者彼此之間的相對距離。用一張標示距離帶的紙會很有幫助。在這個模型中，追擊／逃離動作會讓你相對於追逐中的所有其他參與者一起前進。至於其他動作，你則必須指明要針對哪一名參與者，而所有限制都以你相對於該目標的位置來判定。對所有位在你前方的參與者而言，你都算是追擊者；對所有位在你後方的參與者而言，你都算是獵物。

## 追逐障礙
每一輪中，在參與者秘密選好動作之後，主持人會隨機產生一個追逐障礙。接下來兩頁有兩張追逐障礙表，一張用於徒步追逐，另一張用於地面載具追逐。你可能需要依據遊戲設定調整它們。

障礙除了為追逐增添風味外，也會以不同方式影響參與者所選的動作。主持人可以依現場情況自由調整障礙，甚至在需要時自行設計障礙。

### 從載具上射擊
在追逐中，乘客可以從載具上進行遠程攻擊，而這些攻擊會在駕駛結算完自己的動作後立刻進行。不過，若駕駛執行的不是「停步射擊」，這類攻擊會承受 −2 調整值。

較先進的軍用與警用載具可能配備有瞄準系統的車載武器。這類武器即使駕駛沒有執行停步射擊，也能由砲手在沒有減值的情況下開火。有些更先進的載具甚至配備有自動瞄準系統，擁有自己的射擊技能，並可依駕駛命令開火（不算動作），因此完全不需要砲手。

## 徒步追逐障礙

| D10 | 障礙 |
| --- | --- |
| 1 | 死路：若獵物選擇了追擊／逃離、躲藏或阻擋，該動作會自動失敗。 |
| 2 | 食物攤：一個食物攤或市場攤位擋住去路。若獵物選擇逃離、躲藏或阻擋，必須先進行一次強力擲骰（不算動作）來撞破玻璃。若失敗，該動作取消，且獵物承受 D3 點傷害。 |
| 3 | 車輛／手推車：一輛車或手推車忽然切入，擋住去路。這會讓追擊／逃離與停步射擊獲得 +2 調整值，但攔截承受 −2。 |
| 4 | 人群：街上的人群為獵物提供掩護。躲藏動作獲得 +2 調整值；即使獵物沒有躲藏，追擊者也必須進行一次觀察擲骰（不算動作），若失敗，本輪動作自動失敗。任何失敗的停步射擊都會誤傷路人。 |
| 5 | 僧侶：一大群披著長袍、口中誦唱的僧侶擋住道路。獵物必須擲說服（不算動作）讓他們讓開。若成功，任何躲藏動作都獲得 +2 調整值；若失敗，獵物的動作自動失敗。 |
| 6 | 守衛／警察：一對守衛或警察拔出武器，試圖阻止追逐。這會讓追擊／逃離、躲藏與停步射擊都承受 −2 調整值。任何執行停步射擊的人都會遭到守衛攻擊。 |
| 7 | 老人：一名老人抓住追擊者，喋喋不休地討錢。追擊者必須給錢，或用強力把他推開。若失敗，該追擊者本輪動作自動失敗。 |
| 8 | 垃圾：街道上到處都是垃圾與殘骸，使地面變得濕滑。本輪所有追擊／逃離動作承受 −2，而阻擋獲得 +2。 |
| 9 | 開闊地：追逐者突然來到一片空地，人群暫時散開。這會讓停步射擊獲得 +2 調整值，但躲藏承受 −2。 |
| 10 | 地痞：D6 名街頭混混擋住道路，出言威脅。獵物必須進行說服擲骰（不算動作）才能通過。若失敗，混混就會發動攻擊，所選動作自動失敗，且其中一名混混會立刻發動一次近戰攻擊。若說服成功，混混則會改為攻擊追擊者。 |

## 載具追逐障礙

| D10 | 障礙 |
| --- | --- |
| 1 | 死路：若獵物選擇了追擊／逃離、躲藏或阻擋，該動作會自動失敗。 |
| 2 | 暴雨：突如其來的大雨使視線變差，路面也變得濕滑。追擊／逃離承受劣勢，躲藏則獲得優勢。 |
| 3 | 車輛／手推車：一輛車或手推車突然切入並擋住去路。這會讓阻擋獲得優勢，但追擊／逃離承受劣勢；任何失敗的追擊／逃離都會對駕駛的載具造成 D3 點傷害。 |
| 4 | 紅燈：行人穿越道的號誌轉紅，一大群人開始穿越馬路。這會讓攔截獲得優勢，但追擊／逃離與停步射擊承受劣勢。任何這類失敗動作都會造成波及傷害。 |
| 5 | 巡邏車：一輛警用巡邏車拉著警報器加入追逐。這會讓追擊／逃離、躲藏與停步射擊承受劣勢。任何執行停步射擊的人都會遭警方開火。 |
| 6 | 高速公路：載具駛上寬闊的多線高速公路，交通暫時疏通。這會讓追擊／逃離與停步射擊獲得優勢，但躲藏與阻擋承受劣勢。 |
| 7 | 道路施工：前方有大型道路施工。這會讓躲藏、阻擋與攔截獲得優勢，但追擊／逃離與停步射擊承受劣勢。若追擊／逃離失敗，載具承受 D3 點傷害。 |
| 8 | 單車群：數十名騎士在街上前進，形成吵雜、鏗鏘作響且幾乎無法穿透的屏障。這會讓阻擋獲得優勢，但追擊／逃離與停步射擊承受劣勢。任何這類失敗動作都會造成波及傷害。 |
| 9 | 卡車：一輛巨大的卡車擋住去路。這會讓攔截獲得優勢，但追擊／逃離承受劣勢。若追擊／逃離失敗，載具承受 D6 點傷害。 |
| 10 | 巷道：追逐進入一條狹窄巷弄。在任何動作結算前，必須先進行一次單獨的技能擲骰（不算動作）才能安然通過；若失敗，載具就成為殘骸，追逐也隨之結束。駕駛可以選擇取消自己的動作，改為緩慢而小心地穿過巷道（不需擲骰）。 |
