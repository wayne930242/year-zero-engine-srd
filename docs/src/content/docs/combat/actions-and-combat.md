## 移動
在戰鬥中移動時，你可以花費一個快速動作，從一個區域移動到相鄰區域，或在同一區域內於敵人或玩家角色之間的短程與貼身距離間移動。移動通常不需要擲骰，除非是要進入雜亂區域。

### 匍匐前進
在臥倒狀態下移動，也就是匍匐前進，屬於慢速動作。這表示你在同一輪內不能匍匐前進兩次。在狹窄區域中，匍匐前進是唯一可能的移動方式。

### 近戰接觸中的移動限制
若有活躍敵人處於你的貼身距離，你不能直接走開。你必須改為撤退（見下方）。

### 門與艙口
你可以用一個快速動作打開未上鎖的門或艙口。上鎖的門或艙口可以被強行破壞。一般木門或艙口在屈服前可承受 5 點傷害。更堅固的門需要更大的力量，而且也有護甲值（見[武器、傷害與狀態](/combat/weapons-and-damage/)）。

### 追逐
徒步或使用載具進行追逐時，移動也會以不同方式處理。見[追逐](/combat/chases/)。

### 載具
載具移動有專用規則。更多資訊請見[危害與載具](/combat/hazards-and-vehicles/)。

## 撤退
若有活躍敵人處於你的貼身距離，且你們之間沒有障礙物，你必須進行一次機動擲骰才能遠離對手。若失敗，你仍然會移動，但敵人會立刻對你發動一次免費近戰攻擊。這次免費攻擊在回合順序外發生，而且不會計入他本輪可執行的動作。你不能格擋這次攻擊。

## 伏擊與偷襲
在衝突中取勝的關鍵，往往是在敵人最意想不到的時候出手。你可以用幾種不同的方式做到這件事。

### 偷襲
當你在未被察覺的情況下接近某人並發動攻擊時，這稱為偷襲。你先進行一次潛行擲骰。若要靠近到足以進行近戰攻擊的距離（貼身），則承受 −2 調整值。若你失敗，敵人就會注意到你，接著抽取先攻。

若你成功，你的攻擊視為出其不意，代表你可以自行選擇任何先攻牌。若是近戰偷襲，你還會獲得 +3 調整值，且目標不能格擋這次攻擊。偷襲永遠是個別進行的，也就是一名攻擊者對一個目標。

### 伏擊
偷襲的一種特殊形式是伏擊，也就是埋伏等待敵人經過時出手。此時每名受害者都要進行一次觀察擲骰，以發現伏擊；若攻擊者準備充分，則該擲骰承受 −2 調整值。所有失敗者都會拿到底部的先攻牌（從 #10 往上數），並隨機分配。

## 近戰
當你進行近戰攻擊時，使用近戰技能。近戰發生在與目標貼身距離時。你與目標之間不能有障礙物；若有，你必須先跨越障礙。掩護對近戰攻擊沒有作用。

### 傷害
若你的擲骰成功，你就命中對手，並造成武器的基礎傷害值。每多一個成功，傷害再增加 1。

### 姿態
若你處於臥倒狀態，你的近戰攻擊承受 −2 調整值。起身是快速動作。對臥倒目標的近戰攻擊獲得 +2 調整值。

### 無防備目標
若你攻擊的是無防備或未察覺的目標，你獲得 +3 調整值。即使目標處於臥倒狀態，這個加值也不會進一步提高。

### 特殊攻擊
在近戰中，你有時可能想達成的不只是單純傷害對手。這必須在你擲任何骰子之前宣告，而且只能在徒手戰鬥時進行。
- 你把對手手持的物品奪走。你可以保留該物品，或把它作為攻擊的一部分丟到相鄰區域。
- 你把對手絆倒在地。
- 你把對手推到短程。
- 你把敵人緊緊壓制住。見下文「擒抱」。

### 擒抱
若你成功擒抱對手，你和對手都會倒地。對手會放下手中的武器，而且不能移動。對手唯一能執行的動作是嘗試掙脫，也就是以慢速動作進行一次與你對抗的近戰擲骰，若對手勝出則成功。在你擒抱對手期間，除了放開對手之外，你唯一能執行的動作就是擒抱攻擊。這視為一般徒手攻擊，但屬於快速動作，且不能被格擋。

### 飛撲打擊
若你在同一輪中先移動到敵人的貼身距離，再以徒手攻擊他，你的攻擊獲得 +2 調整值。這稱為飛撲打擊。若攻擊命中，你照常造成傷害，且你與目標都會倒地。若攻擊失敗，則只有你自己跌倒。

## 格擋
當敵人以近戰攻擊你時，你可以擲近戰來格擋攻擊，以避免被命中。這是快速動作。你必須在攻擊者為其攻擊擲骰之前，先聲明你要嘗試格擋。這會讓該次攻擊實際上變成一次對抗擲骰（見[裝備與技能列表](/skills-and-specialties/gear-and-skill-list/)）。你每擲出一個成功，就能抵消攻擊者擲骰中的一個成功。多出的成功沒有額外效果。

### 徒手格擋
如果你沒有持武器，你只能格擋其他人形生物的徒手攻擊。若要格擋持武器的近戰攻擊，或動物、野獸之類的攻擊，你必須持有某種堅固的武器或工具。

### 動作消耗
格擋會打斷回合順序，但它仍會計入你本輪可用的兩個動作（一慢一快）。你每執行一次格擋，等到輪到你行動時，就少一個可用動作；一旦你已在本輪中用掉兩個動作來格擋，你就不能再繼續格擋更多攻擊。每輪開始時，所有動作都會刷新。你可以保留尚未使用的動作，留到本輪稍後拿來格擋，但不能從一輪保留到下一輪。

### 近戰調整值

| 因素 | 調整值 |
| --- | --- |
| 攻擊者臥倒 | −2 |
| 目標臥倒 | +2 |
| 無防備目標 | +3 |
| 飛撲打擊 | +2 |

## 遠程戰鬥
若要用手持遠程武器攻擊某人，請擲射擊；投擲武器則使用機動。拔出武器是快速動作，開火則是慢速動作。

### 傷害
若你的攻擊成功，你就命中目標，並造成武器的基礎傷害值。每多一個成功，傷害增加 1。

### 距離
武器表會標示各武器的射程，也就是該武器能使用的最遠距離類別。目標越遠，就越難命中。中程承受 −1 調整值，長程承受 −2。貼身距離則承受 −3，因為這麼近很難瞄準對手。若你射擊的是無防備或毫無警覺的敵人，則不受這項減值，反而改為獲得 +3 調整值。

### 目標尺寸
射擊大型目標，例如載具，攻擊獲得 +2 調整值。射擊小型物體，例如艙口或手持物品，則承受 −2 調整值。

### 未見目標
若你看不見目標（因黑暗或掩護），但仍大致知道其位置，你仍可射擊，但承受 −2 調整值。

### 瞄準射擊
若你在扣下扳機前花時間仔細瞄準，你的攻擊擲骰獲得 +2。瞄準是快速動作。若你在瞄準後，除了射擊武器之外還做了其他任何事，或者你受到傷害，你就會失去瞄準效果，必須再花一個快速動作重新瞄準。

### 望遠瞄具
若你的武器裝有望遠瞄具，你可以用慢速動作瞄準，並再額外獲得 +1 加值（總共 +3）。請注意，你不能在同一輪中同時用慢速瞄準並開火。

### 遠程射擊調整值

| 因素 | 調整值 |
| --- | --- |
| 瞄準射擊 | +2 |
| 目標貼身 | −3/+3 |
| 短程 | — |
| 中程 | −1 |
| 長程 | −2 |
| 極遠程 | −3 |
| 大型目標 | +2 |
| 小型目標 | −2 |
| 昏暗 | −1 |
| 黑暗 | −2 |

## 彈藥
火器的彈藥可以有不同處理方式，取決於遊戲設定的科技水準，以及你想要的複雜度。你可以精確計算每一發子彈、透過補給擲骰處理彈藥（見[專長、裝備與經驗](/player-characters/specialties-gear-and-experience/)），或使用更詳盡的彈藥骰系統（見下方側欄）。簡單的單發武器每次射擊後都必須重新裝填。裝填火器屬於慢速動作。

### 弓與彈弓
弓與彈弓並不是真的需要「重新裝填」；你必須改為花費一個快速動作，替武器做好準備，例如搭箭或把石頭放進彈弓。當你完成準備後，除了射擊或瞄準（見上文）之外，不能做其他事；若你做了別的事，在再次射擊前就必須重新準備武器。十字弓則不需要這樣準備。它可以像火器一樣在已上弦狀態下攜帶，而上弦屬於慢速動作。

## 全自動射擊
某些武器能進行全自動射擊。是否使用相關規則，甚至是否要為此設計專門規則，取決於你想要的複雜度。以下提供一個基本版本。更複雜的規則請見彈藥骰側欄。

### 基本全自動
一次全自動點射視為一般遠程攻擊，但如果你命中，可以立刻再對同一目標或不同目標發動一次攻擊。若再度命中，你可以進行第三次攻擊。即使第三次也命中，你也不會獲得更多攻擊。待所有攻擊結算完畢後，你必須為彈藥進行一次補給擲骰（見[專長、裝備與經驗](/player-characters/specialties-gear-and-experience/)）。

### 彈藥骰

若你希望火器管理更細緻、更複雜，可以使用彈藥骰系統。彈藥骰一律使用 D6，並應與基礎骰使用不同顏色。在此系統中，所有火器都有射速（RoF）值，從一般手槍的 2，到全自動武器的 5 或 6。

你可以在攻擊中加入任意數量的彈藥骰，從 0 顆到不高於射速，或不高於彈匣剩餘子彈數減 1，兩者取其低。舉例來說，如果你的彈匣剩 3 發子彈，你最多可加入 2 顆彈藥骰。若只剩 1 發，則不能加入任何彈藥骰。在這種情況下，或你選擇不使用彈藥骰時，該次攻擊只會消耗 1 發彈藥。

彈藥骰擲出 6 時，和 D6 基礎骰一樣算作成功，但彈藥骰上的成功不會用來判定是否命中目標。相反地，若你的攻擊命中（只檢查基礎骰），每個彈藥骰成功都可以用來讓傷害提高一級，或對同一目標、或同區域中的第二目標觸發一次額外命中。每次額外命中都造成基礎傷害，並且每再多花一個成功，該次命中的傷害便再增加 1。

擲骰後，把所有彈藥骰的 D6 點數加總，然後再加 1。這個總和就是你這次攻擊消耗的彈藥總數。若你彈藥骰的總和等於或超過彈匣中剩餘的子彈數，你的彈匣就被完全打空。

## 戒備射擊
只要你手上持有遠程武器，且沒有敵人處於貼身距離，你就可以用一個快速動作朝指定方向進入戒備射擊姿態。這表示你正朝該方向瞄準並準備射擊。從你進入戒備射擊姿態開始，到下一輪輪到你行動之前，你都可以朝所選方向上的目標開火。

你可以在回合順序中的任何時點開火，而且你的射擊會先於所有其他動作結算，即使那些動作已經宣告也是如此。例如，若你瞄準方向上的某名敵人宣告要射擊武器，你可以先開槍。敵人在遭到你的戒備射擊攻擊後，不能更改自己的攻擊。

在戒備射擊姿態下開火，算作一般攻擊（慢速動作）。因此，若你想在戒備射擊中開火，就必須保留本輪的慢速動作用於那次攻擊。
若你和某名敵人互相對彼此進入戒備射擊姿態，且都選擇朝彼此開火，則以一次對抗的射擊擲骰決定哪一方先開火。這次擲骰不計入任何一方的動作。

### 失去戒備射擊
你會一直維持戒備射擊姿態，直到你自行解除，或發生以下任一情況：
- 你執行了戒備射擊開火以外的任何動作。
- 你遭受近戰攻擊。
- 你受到傷害。

武器欄位的詳細說明與武器表，請見[武器、傷害與狀態](/combat/weapons-and-damage/)。
